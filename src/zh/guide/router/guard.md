# 路由守卫

## 守卫流程

```
页面跳转 → 常量路由？→ 放行
         → 已登录？ → 否 → 登录页
                    → 是 → 路由已加载？→ 加载 → 有权限？→ 放行
                                                      → 403
```

## 守卫文件

- `route.ts` — 认证、动态路由加载、权限检查
- `progress.ts` — NProgress 加载进度条
- `title.ts` — 页面标题同步

## 常量路由

`constant: true` 的路由跳过所有认证检查，用于登录页、错误页等。
