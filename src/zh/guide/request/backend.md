# 对接后端

## Token 处理

请求拦截器自动添加 JWT Token 到每个请求的 Authorization 头。

## Token 自动刷新

后端返回 `2103` 时，前端自动使用 refresh token 刷新并重试原始请求。

## 错误码映射

| 码值 | 行为 |
|------|------|
| `0000` | 成功，提取数据 |
| `2100`, `2101` | 跳转登录页 |
| `2102` | 弹窗确认后登出 |
| `2103` | 自动刷新 Token |
| 其他 | 显示错误消息 |

## 新增 API

1. 在 `service-alova/api/` 定义函数
2. 在 `typings/api.d.ts` 定义类型
3. 在组件或 Store 中使用
